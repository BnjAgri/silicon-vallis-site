# Silicon Vallis — site vitrine (statique)

Site vitrine / portfolio statique (HTML/CSS/JS) pour présenter Silicon Vallis et promouvoir Loc Vallis.


```bash
python3 -m http.server 5173
```

Puis ouvrir http://localhost:5173
